# Install Zola locally [here](https://www.getzola.org/documentation/getting-started/installation/)


# Quick Start:

## Downlaod the source repo
```bash
git clone https://github.com/spacecubics/www.git
cd www
```
## Build site and serve locally
```bash
zola serve
```
